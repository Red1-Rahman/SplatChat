# SplatChat   
### An AI-powered 3D world viewer where natural language chat controls camera movements around a Gaussian Splat model.


Try messages like:   

  ```Show me the side```   
  ```Can I see the top view?```   
  ```Show me a detail```   

---

### Target: RPG games, Interactive education etc
### Future Plans: voice Ai for accessibility
